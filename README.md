# People Analytics Aplicado a la Predicción de Rotación Laboral

Proyecto desarrollado como trabajo final de la Licenciatura en Ciencia de Datos en UCASAL.

## Descripción

El objetivo principal de este proyecto fue desarrollar un modelo predictivo capaz de identificar colaboradores con mayor probabilidad de abandonar una organización, aplicando técnicas de People Analytics y Machine Learning.

El proyecto busca aportar una herramienta de apoyo a la toma de decisiones estratégicas en Recursos Humanos, permitiendo detectar posibles casos de rotación laboral de manera anticipada y facilitar acciones preventivas orientadas a la retención de talento.

---

## Objetivos

- Analizar variables relacionadas con rotación laboral
- Aplicar técnicas de Machine Learning
- Comparar distintos modelos predictivos
- Evaluar desempeño mediante métricas
- Proponer una implementación práctica para RRHH

---

## Metodología

El proyecto fue desarrollado utilizando la metodología CRISP-DM:

1. Comprensión del negocio
2. Comprensión de los datos
3. Preparación de datos
4. Modelado
5. Evaluación
6. Implementación conceptual

---

## Modelos utilizados

- Regresión Logística
- Árbol de Decisión
- Random Forest
- XGBoost

---

## Resultado destacado

El modelo XGBoost presentó el mejor desempeño general, logrando un equilibrio sólido entre precisión y capacidad de detección de casos de riesgo de rotación laboral.

Se planteó una aplicación en Streamlit orientada a profesionales de Recursos Humanos para facilitar el análisis y visualización de posibles riesgos de rotación laboral.
